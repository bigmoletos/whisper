# Changelog - Voice-to-Text Basic

Toutes les modifications notables de ce projet sont documentées dans ce fichier.

## [1.0.0] - 2026-01-20

### Ajouté
- Transcription vocale en temps réel avec Whisper
- Support des modèles tiny, base, small, medium
- Raccourcis clavier configurables
- Collage automatique du texte transcrit
- Notifications système Windows
- Support multilingue (français par défaut)
- Configuration via fichier JSON

### Technique
- Restructuration du projet en sous-modules
- Séparation du code partagé
- Scripts de lancement simplifiés
